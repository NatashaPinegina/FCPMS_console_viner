# FCPMS_console_viner
#### Программа предназначена для оценки навигационных параметров множественных источников радиоизлучения на основе алгоритма винеровской фильтрации с цифровой предобработкой.

### Сборка
```
git clone 
cd FCPMS_console_viner
mkdir build
cd build
cmake  ../
make
./console_FCPMS
```
